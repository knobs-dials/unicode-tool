http://www.unicode.org/Public/security/

This public directory contains data files for the UTS #39: Unicode Security Mechanisms.

The subdirectories contain data files associated with a given version, with names such as:

    http://www.unicode.org/Public/security/revision-04/
    http://www.unicode.org/Public/security/6.3.0/

The data files for the latest approved version are also in the directory:

    http://www.unicode.org/Public/security/latest 

For more information, see http://www.unicode.org/reports/tr39/.
